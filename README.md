# NFT Art Generator
Código para gerar desenhos e nomes para um NFT randomico, com um número limitado de quantos podem ser criados.

!!! Este é um código criado com a ajuda do canal Fireship https://www.youtube.com/watch?v=meTpMP0J5E8 !!!

## Usage

```
git clone <this-repo>
npm install
node .
```
